Features

WYSIWYG Editing

Type markdown and see it rendered in real time. No split pane, no preview toggle. What you see is what you get.

Light & Dark Themes

Switch between themes with a single click. Each carefully tuned for readability and comfort.

Code & Math

Syntax-highlighted code blocks via CodeMirror. Inline and block LaTeX equations powered by KaTeX.

File Management

Built-in sidebar with file tree, quick open palette, and drag & drop. Your workspace, organized.

Tables & Diagrams

Full table support with inline editing. Mermaid diagrams rendered right inside the editor.

Auto Save

Changes saved automatically as you type. Never lose your work again.
